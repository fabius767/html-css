O que e box model?

No desenvolvimento da Web, o modelo de caixa CSS refere-se a como os elementos HTML são modelados nos mecanismos do navegador e como as dimensões desses elementos HTML são derivadas das propriedades CSS. É um conceito fundamental para a composição de páginas web em HTML.
